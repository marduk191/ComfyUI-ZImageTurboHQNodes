# ComfyUI ZImage Turbo HQ Nodes

Self-contained custom nodes for Tongyi Z-Image-Turbo. Note that these have no real settings other than presets. They are easy mode quick nodes.

## Included nodes
- ZImageTurboConditioning
- ZImageTurboLatentInit
- ZImageTurboSeedControl
- ZImageTurboSamplingPlan
- ZImageTurboSampler
- ZImageTurboTwoPassRefiner

## Notes
- Registers local scheduler aliases: `capitanZiT`, `zimage_turbo`
- Registers local sampler: `euler_flow`
- No external Capitan repos required
